⚙️ Avatim Gestão — Relatório de Correções

15 bugs identificados e corrigidos · Patch pronto para aplicar

🐛 Bugs Encontrados

1maskBR não definidaCRÍTICO
A função maskBR() é chamada em +20 inputs de formulário (metas, campanhas, vendas) mas nunca foi implementada no código. Todos os campos monetários não funcionam — nenhum dado de meta ou campanha pode ser salvo corretamente.
2selecionarProdCamp não definidaCRÍTICO
Botão "+Camp" na tabela de produtos chama selecionarProdCamp() que não existe → ReferenceError ao tentar adicionar qualquer produto a uma campanha.
3IDs de filtros de campanha duplicadosCRÍTICO
O HTML tem dois blocos com os mesmos IDs (camp-fil-todos, camp-fil-NATAL, etc.). O DOM sempre usa o primeiro elemento encontrado, tornando o segundo bloco de filtros completamente inoperante. Cria comportamento imprevisível ao chamar filtrarCampTipo().
4#camp-lista duplicadoCRÍTICO
Há dois elementos com id="camp-lista" no HTML. renderCampLista() usa document.getElementById() que retorna apenas o primeiro — a lista de campanhas pode ser renderizada no elemento errado (fora da aba de campanhas).
5tabCfg declarada duas vezesAVISO
A função tabCfg() existe em dois blocos <script> separados com corpos idênticos. Em JS strict isso seria um erro; no modo normal a segunda sobrescreve a primeira sem efeito visível, mas é um risco de manutenção.
6saveWithSync declarada duas vezesCRÍTICO
Há uma versão simples de saveWithSync() perto do topo do script e uma versão completa (com Supabase/Firebase) mais abaixo. A segunda sobrescreve a primeira, mas a simples era chamada na lógica de salvamento de lançamentos — resultado: dados de campanha e lançamentos podem não sincronizar corretamente.
7D._user inexistente (referência incorreta)AVISO
Em saveWithSync(), o código usa D._user?.email para registrar quem lançou. Porém D._user nunca é definido — o usuário correto está em window._currentUser. Resultado: campo lancado_por fica vazio em todos os registros.
8COSMOS_CONFIG.containers indefinidoAVISO
cosmosRequest() acessa COSMOS_CONFIG.containers[container] mas o objeto COSMOS_CONFIG não tem essa propriedade → TypeError: Cannot read properties of undefined ao tentar usar qualquer função do Cosmos DB.
9sincronizarCosmosParaLocal não definidaAVISO
Chamada dentro de testarCosmos() mas nunca implementada → ReferenceError ao testar conexão Cosmos DB bem-sucedida.
10preencherFBForm vs preencherFormFB — nomes inconsistentesAVISO
A função é definida como preencherFBForm() mas chamada em tabCfg() como preencherFormFB() → aba Firebase na tela de configurações nunca preenche os campos ao abrir.
11_tabVend / _tabMeta sem inicialização garantidaMENOR
Variáveis de estado de aba usadas antes de garantidamente serem definidas em alguns paths de navegação direta. Pode causar comportamento inesperado em links diretos para páginas.
12calcMetasAuto sem proteção contra divisão por zeroMENOR
Se diasDoMes() retornar 0 ou um valor inválido (ex: bug de fuso horário no último dia do mês), meta/DU resulta em Infinity ou NaN, corrompendo todas as metas exibidas.
13getMetas sem try/catch — propaga erros silenciososMENOR
Se o dado salvo no localStorage estiver corrompido para uma loja específica, getMetas() propaga o erro para todos os renderizadores que a chamam, quebrando o dashboard inteiro.
14renderCampLista sem verificação de D.campObjetivosMENOR
Se chamada antes de initCampData() (ex: navegação direta), acessa D.campObjetivos.filter em undefined → tela de campanhas quebra com tela branca.
15D._campExcluidos pode ser undefinedMENOR
Referenciado em múltiplos lugares como D._campExcluidos || [] mas a inicialização no topo do script é feita corretamente — porém se o localStorage tiver um snapshot antigo sem esse campo, o fallback para [] no código Firebase não é sempre aplicado.

✅ Como Aplicar o Patch

Há duas formas de aplicar as correções:

1
Abra o arquivo HTML do sistema no editor de texto (VS Code, Notepad++, etc.)
2
Localize a tag de fechamento </body> perto do final do arquivo (última linha antes de </html>)
3
Cole todo o bloco <script> abaixo imediatamente antes de </body>
4
Salve o arquivo e recarregue no browser com Ctrl+Shift+R (hard reload sem cache)

// ── Bug 1: maskBR ausente
window.maskBR = function(el) {
  let v = el.value.replace(/[^\d]/g, '');
  if (!v) { el.value = ''; return; }
  v = (parseInt(v, 10) / 100).toFixed(2);
  el.value = Number(v).toLocaleString('pt-BR', { minimumFractionDigits: 2, maximumFractionDigits: 2 });
};

// ── Bug 2: selecionarProdCamp ausente
window.selecionarProdCamp = function(cod, nome) {
  if (typeof abrirFormCampObj === 'function') {
    const f = document.getElementById('form-camp-obj');
    if (f && f.style.display === 'none') abrirFormCampObj();
  }
  if (typeof adicionarProdCampObj === 'function') {
    adicionarProdCampObj(cod, nome, '');
  } else {
    if (!window._campProdsObj) window._campProdsObj = [];
    if (!window._campProdsObj.find(p => p.c === cod)) {
      window._campProdsObj.push({ c: cod, n: nome, g: '' });
      if (typeof renderProdsSelecionados === 'function') renderProdsSelecionados();
    }
  }
  showToast('Produto adicionado à campanha!');
};

// ── Bug 3+4: IDs duplicados (#camp-lista e #camp-fil-*)
document.addEventListener('DOMContentLoaded', function() {
  // Remover elementos duplicados de filtros de campanha fora de #aba-camp-content
  const seen = new Set();
  document.querySelectorAll('[id^="camp-fil-"]').forEach(el => {
    if (seen.has(el.id)) {
      if (!el.closest('#aba-camp-content')) el.remove();
    } else { seen.add(el.id); }
  });
  // Remover #camp-lista duplicado fora de #aba-camp-content
  const campListas = document.querySelectorAll('#camp-lista');
  if (campListas.length > 1) {
    campListas.forEach(el => { if (!el.closest('#aba-camp-content')) el.remove(); });
  }
  // Bug 11: garantir variáveis de estado
  if (typeof window._tabVend === 'undefined') window._tabVend = 'dia';
  if (typeof window._tabMeta === 'undefined') window._tabMeta = 'dia';
  if (typeof window._mesMeta === 'undefined') window._mesMeta = '';
  if (typeof window._rankVendOrdem === 'undefined') window._rankVendOrdem = 'fat';
  // Bug 8: COSMOS_CONFIG.containers
  if (typeof COSMOS_CONFIG !== 'undefined' && !COSMOS_CONFIG.containers) COSMOS_CONFIG.containers = {};
  // Bug 15: D._campExcluidos
  if (window.D && !D._campExcluidos) D._campExcluidos = [];
  console.log('[Avatim] ✅ Patch aplicado.');
});

// ── Bug 7: D._user → window._currentUser
const _origSaveWithSync = window.saveWithSync;
window.saveWithSync = function(opts) {
  if (window._currentUser && window.D && !D._user)
    D._user = { email: window._currentUser.email };
  if (typeof _origSaveWithSync === 'function') return _origSaveWithSync(opts);
  localStorage.setItem('avatim_v5', JSON.stringify(D));
};

// ── Bug 9: sincronizarCosmosParaLocal ausente
window.sincronizarCosmosParaLocal = async function() {
  if (!window.fsCarregar) return;
  try {
    const json = await window.fsCarregar();
    if (!json) return;
    const remoto = JSON.parse(json);
    if (remoto.vendas) Object.entries(remoto.vendas).forEach(([lid, dias]) => {
      if (!D.vendas[lid]) D.vendas[lid] = {};
      Object.assign(D.vendas[lid], dias);
    });
    localStorage.setItem('avatim_v5', JSON.stringify(D));
    showToast('Sincronizado com Firebase!');
  } catch(e) { console.warn('[Avatim] sincronizarCosmosParaLocal:', e.message); }
};

// ── Bug 10: preencherFormFB → preencherFBForm
window.preencherFormFB = function() {
  if (typeof preencherFBForm === 'function') preencherFBForm();
};

// ── Bug 12: calcMetasAuto sem proteção contra divisão por zero
const _origCalcMetasAuto = window.calcMetasAuto;
window.calcMetasAuto = function(mes, ticket, superMes) {
  mes = Number(mes) || 0;
  if (mes <= 0) return { mes: 0, sem: 0, dia: 0, ticket: 150, superMes: 0, superSem: 0, superDia: 0, du: 30, sw: 4.33 };
  superMes = Number(superMes) || parseFloat((mes * 1.255).toFixed(2));
  ticket = Number(ticket) || 150;
  let mr; try { mr = mesAtual(); } catch(e) { mr = new Date().toISOString().slice(0,7); }
  let DU; try { DU = diasDoMes(mr); } catch(e) { DU = 30; }
  if (!DU || DU <= 0) DU = 30;
  const SW = 4.33;
  return {
    mes, ticket, superMes, du: DU, sw: SW,
    dia:      parseFloat((mes / DU).toFixed(2)),
    sem:      parseFloat((mes / SW).toFixed(2)),
    superDia: parseFloat((superMes / DU).toFixed(2)),
    superSem: parseFloat((superMes / SW).toFixed(2))
  };
};

// ── Bug 13: getMetas com try/catch
const _origGetMetas = window.getMetas;
window.getMetas = function(lid, mesRef) {
  try {
    return typeof _origGetMetas === 'function'
      ? _origGetMetas(lid, mesRef)
      : { mes: 0, dia: 0, sem: 0, superMes: 0, superDia: 0, superSem: 0, ticket: 150 };
  } catch(e) {
    console.warn('[Avatim] getMetas erro:', lid, e.message);
    return { mes: 0, dia: 0, sem: 0, superMes: 0, superDia: 0, superSem: 0, ticket: 150 };
  }
};

// ── Bug 14: renderCampLista sem verificação
const _origRenderCampLista = window.renderCampLista;
window.renderCampLista = function() {
  if (window.D) {
    if (!D.campObjetivos) D.campObjetivos = [];
    if (!D.campRealizados) D.campRealizados = [];
  }
  if (typeof _origRenderCampLista === 'function') _origRenderCampLista();
};
